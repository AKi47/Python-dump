You need the pygame module installed
Made for pygame 1.9.4


Files:
Default 1,2,3,5,6,9 are files with walls
Files 4,7,8 are empty

Download all the files into one directory and run version 4 or 5 to work

If you want to start the program without any levels, delete all the text files in "Walls" and then change the number in counter to 1
